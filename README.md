## Название и описание проекта
**Программный комплекс автоматизированного оптического контроля качества печатных плат на базе архитектуры YOLOv11**

Программное решение предназначено для автоматического обнаружения и классификации визуальных топологических дефектов печатных плат в режиме реального времени. 

Модель обучена на распознавание 6 классов производственного брака:
* Отсутствие отверстия (missing hole)
* «Мышиный укус» / скол (mouse bite)
* Обрыв проводника (open circuit)
* Короткое замыкание (short circuit)
* Заусенец (spur)
* Остаточная медь (spurious copper)

**В состав проекта входят:**
1. Обученная нейросетевая модель YOLOv11s, адаптированная для поиска микродефектов.
2. Интерактивный симулятор конвейерной ленты на базе библиотеки Pygame для демонстрации работы системы в динамике.

## Установка зависимостей

Проект разделен на два логических блока: нейросетевое ядро и графический симулятор конвейера. Установите зависимости последовательно:

1. **Установка базовых библиотек для работы YOLO11:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Установка библиотек для графического симулятора**
   ```bash
   pip install -r requirements-conveyor.txt
   ```
Для запуска симулятора конвейера с визуализацией детекции дефектов в реальном времени выполните следующую команду из корневой папки проекта:
  ```bash
  python conveyor_demo.py --source .\pcb_original_image --weights .\runs\detect\train-6\weights\best.pt --conf 0.2 --loop
  ```

Управление симулятором (Горячие клавиши):
* [ P ] — Пауза / Возобновление движения конвейера.
* [ [ ] / [ ] ] — Уменьшить / Увеличить скорость конвейерной ленты.
* [ L ] — Зациклить подачу плат из папки.
* [ ЛКМ по логу ] (Traceability Log) — Открыть архивный снимок проинспектированной платы.
* [ Esc ] — Закрыть просмотр фото / Выйти из программы.
